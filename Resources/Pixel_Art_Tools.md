# Pixel Art & Development Resources

## Libraries & Frameworks

### JavaScript / Game Dev
- **[Phaser](https://phaser.io/)**: excellent for 2D games, handles sprites and animations easily. Good if the character will walk around.
- **[SpriteJS](https://spritejs.org/)**: Cross-platform graphics system, supports layering.
- **[PixiJS](https://pixijs.com/)**: Fast 2D rendering engine, good for compositing many layers.

### Pixel Art Generation / Avatars
- **[DiceBear](https://www.dicebear.com/)**: Open source avatar library. Has pixel art styles. Good reference for API/structure.
- **[8-biticon](https://8biticon.com/)**: Conceptual reference for "maker" interfaces.

## Tools
- **[Aseprite](https://www.aseprite.org/)**: The industry standard for creating pixel art and animations.
- **[Piskel](https://www.piskelapp.com/)**: Free online pixel art editor.
- **[Figma](https://www.figma.com/)**: (Mentioned in Humation) - Good for designing the *UI* of the character creator.
